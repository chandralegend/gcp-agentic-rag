Prior Authorization Check
by Code

While using the Prior Authorization Check by Code
within Eligibility and Benefits, you'll now see the
option to **Print/Generate a Reference
Number** in the upper right corner of the
window.

* + Select **Yes** when asked
    *Would you like to generate a
    Reference Number?*.
  + Click to include **Code Lookup and
    Selections** and/or
    **Inquiry Response**.
  + Choose to **Print** or
    **Save as PDF**.
